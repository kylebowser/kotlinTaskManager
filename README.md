# Overview

This program is a basic Task Manager application run on the command line. The program has 5 options which are to add a task, remove a task, display all tasks, load a list of tasks from a file, and save all tasks to a file. This allows users to go through the full process of deciding on a task, recording it, saving it, leaving to complete the task, coming back and loading the task, and then removing the task from their list. This ecapsulates the basic experiance that one would want from a task manager. 

The purpose of this program was to give myself a practice program for my learning of the Kotlin programing language. It allowed me to put what I was learing into practice without a tutorial holding my hand.
